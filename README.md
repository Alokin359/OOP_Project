# 📝 Мениджър на задачи и проекти (CLI)

Конзолно приложение на C++ за управление на ежедневни задачи и по-големи проекти. Проектът демонстрира основни принципи на Обектно-ориентираното програмиране (ООП) като **наследяване, капсулация и полиморфизъм**, както и работа с файлове.

---

## 🚀 Налични функционалности (Чекпойнт 2)

1. **Създаване на задачи:** Поддържат се обикновени задачи (`SimpleTask`) и проекти с подзадачи (`ProjectTask`) чрез ООП йерархия.
2. **Преглед на списъка:** Показва ID, заглавие, тип и автоматично изчислен прогрес (в %).
3. **Управление на подзадачи:** Възможност за динамично добавяне на подзадачи към проект и автоматично обновяване на общия му прогрес.
4. **Персистентност (Файлове):** Автоматично записване и зареждане на данните от текстовия файл `tasks.txt`.

---

## 🛠️ Как се борави с менюто

При стартиране на програмата изберете опция от `1` до `7`:

* **`1` (Нова задача):** Въведете уникално ID (без интервали) и заглавие на задачата (напр. `T1` -> `Да купя хляб`).
* **`2` (Нов проект):** Въведете ID и име на проекта (напр. `P1` -> `Курсова работа`). Започва автоматично от 0% прогрес.
* **`3` (Списък):** Принтира всички въведени задачи и техния текущ прогрес в конзолата.
* **`4` (Добави подзадача):** Въведете ID на съществуващ проект и име на новата подзадача към него.
* **`5` (Завършване):** Въведете ID на задача. 
  * Ако е `SimpleTask`, прогресът става 100%. 
  * Ако е `ProjectTask`, въведете номера на подзадачата (започвайки от `0`), която сте изпълнили.
* **`6` (Запис):** Ръчно записва текущото състояние във файла `tasks.txt`.
* **`7` (Изход):** Автоматично записва промените и затваря програмата. При следващ старт данните се зареждат сами.

## 🤖 Автоматични тестове

За да не въвеждате команди ръчно в конзолата всеки път, е подготвен скрипт за автоматично тестване чрез пренасочване на входния поток (`Input Redirection`).

1. Уверете се, че имате файл `tests.txt` със записани тестови команди в директорията.
2. Стартирайте теста със следната команда в терминала:

```bash
g++ -o task_manager main.cpp
chcp 65001
task_manager < tests.txt
