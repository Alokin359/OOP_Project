# Мениджър на задачи и проекти (CLI)

Конзолно приложение на C++ за управление на ежедневни задачи и по-големи проекти. Проектът демонстрира основни принципи на Обектно-ориентираното програмиране (ООП) като наследяване, капсулация и полиморфизъм, както и персистентност на данните чрез работа с файлове.

---

## Налични функционалности

1. Създаване на задачи: Поддържат се обикновени задачи (SimpleTask) и по-комплексни проекти с подзадачи (ProjectTask) чрез ООП йерархия.
2. Преглед на списъка: Показва ID, заглавие, тип на задачата и автоматично изчислен прогрес (в %).
3. Управление на подзадачи: Възможност за динамично добавяне на подзадачи към проект и автоматично обновяване на общия му прогрес спрямо завършените подзадачи.
4. Персистентност (Файлове): Автоматично записване и зареждане на данните от текстовия файл tasks.txt, като се запазва пълната структура на проектите и техните подзадачи.

---

## Как се борави с менюто

При стартиране на програмата изберете опция от 1 до 7:

* 1 (Нова задача): Въведете уникално ID (без интервали) и заглавие на задачата (напр. T1 -> Да купя хляб).
* 2 (Нов проект): Въведете ID и име на проекта (напр. P1 -> Курсова работа). Започва автоматично от 0% прогрес.
* 3 (Списък): Принтира всички въведени задачи и техния текущ прогрес в конзолата.
* 4 (Добави подзадача): Въведете ID на съществуващ проект и име на новата подзадача към него.
* 5 (Завършване): Въведете ID на задача. 
  * Ако е SimpleTask, прогресът става 100%. 
  * Ако е ProjectTask, въведете номера на подзадачата (започвайки от 0), която сте изпълнили.
* 6 (Запис): Ръчно записва текущото състояние във файла tasks.txt.
* 7 (Изход): Автоматично записва промените и затваря програмата. При следващ старт данните се зареждат сами от файла.

## Компилиране и Стартиране

Изберете инструкциите спрямо вашата операционна система:

### За macOS / Linux (Терминал)
Отворете терминала в папката на проекта и изпълнете:
```bash
g++ -std=c++17 main.cpp -o task_manager
./task_manager
```

### За Windows (CMD / PowerShell)
Отворете командния ред в папката на проекта и изпълнете:
```bash
g++ -std=c++17 main.cpp -o task_manager
task_manager
```

## Автоматични тестове
За да не въвеждате команди ръчно в конзолата всеки път, е подготвен файл за автоматично тестване чрез пренасочване на входния поток (Input Redirection).
Уверете се, че имате файл tests.txt със записани тестови команди в директорията.
Изпълнете съответната команда според платформата:
### За macOS / Linux:
```bash
./task_manager < tests.txt
```
### За Windows (за правилно разчитане на кирилицата в CMD първо се задава UTF-8 режим):
```bash
chcp 65001
task_manager < tests.txt
```
